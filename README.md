# Vue Infinite Scroll

這是一個基於 Vue 3 和 TypeScript 的無限滾動 (Infinite Scroll) 專案。
為了展示無限滾動的效果，此範例專案會動態載入並顯示 GitHub 用戶 [barchart](https://github.com/barchart) 的公開儲存庫 (repositories) 列表。

[**Live Demo**](https://samwang-infinite-scroll.netlify.app)
